Analiza statystyczna Projekt Portfelowy

Wprowadzenie

Celem projektu jest analiza danych z czterech eksperymentw, sprawdzenie hipotez statystycznych i wycignicie wnioskw biznesowych. Analiza obejmuje testy t-Studenta, Wilcoxona i MannWhitney oraz wizualizacje danych.

# Import bibliotek i wczytanie danych

import pandas as pd
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt
import seaborn as sns

# Wczytanie danych

df1 = pd.read_csv('data/hw1.csv')
df2 = pd.read_csv('data/hw2.csv')
df3 = pd.read_csv('data/hw3.csv')
df4 = pd.read_csv('data/hw4.csv')

Zadanie 1 Porwnanie wynikw nauki (X vs Y)

Hipotezy

H: rednie wyniki grup X i Y s takie same.

H: rednie wyniki grup X i Y rni si.

In[3]:
#Sprawdzenie normalnoci
print("Test Shapiro-Wilka  Grupa X:", stats.shapiro(df1.X))
print("Test Shapiro-Wilka  Grupa Y:", stats.shapiro(df1.Y))

Test Shapiro-Wilka  Grupa X: ShapiroResult(statistic=np.float64(0.984373122264943), pvalue=np.float64(0.28594616875713735))
Test Shapiro-Wilka  Grupa Y: ShapiroResult(statistic=np.float64(0.9864255697754216), pvalue=np.float64(0.4001688833356286))
In[4]:
#Test statystyczny
t_stat, p_val = stats.ttest_ind(df1.X, df1.Y, equal_var=False)
print("t-stat:", t_stat, "p-value:", p_val)

if p_val < 0.05:
    print("Odrzucamy H0  rednie rni si")
else:
    print("Brak podstaw do odrzucenia H0  rednie s podobne")

t-stat: -0.6292883911384963 p-value: 0.5300187896437802
Brak podstaw do odrzucenia H0  rednie s podobne

Wizualizacja

In[5]:
plt.figure(figsize=(8,5))
sns.boxplot(data=df1[['X','Y']])
plt.title("Porwnanie wynikw grup X i Y")
plt.ylabel("Wynik testu")
plt.show()

plt.figure(figsize=(8,5))
sns.histplot(df1.X, kde=True, color='blue', label='X', alpha=0.5)
sns.histplot(df1.Y, kde=True, color='red', label='Y', alpha=0.5)
plt.title("Rozkad wynikw grup X i Y")
plt.legend()
plt.show()

Komentarz biznesowy: Wyniki pokazuj, e rednie wyniki grup nie rni si istotnie. Oznacza to, e obie metody nauki X i Y s rwnie skuteczne.
